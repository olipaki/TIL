1. 함수(Function) 관련 핵심 용어
함수 정의 키워드: def. 함수를 만들 때 가장 먼저 쓰는 명령어입니다.

매개변수 (Parameter): 함수를 정의할 때 입력값을 받기 위해 사용하는 변수의 이름입니다.

인자 (Argument): 함수를 실제로 호출할 때 괄호 안에 전달하는 실제 값입니다.

반환값 (Return Value): 함수가 실행을 마치고 돌려주는 결과값입니다. 키워드는 **return**입니다.

내장 함수 (Built-in Function): print(), len()처럼 파이썬에 이미 만들어져 있는 함수입니다.

메서드 (Method): list.append()처럼 특정 자료형에 속해 있어 **점(.)**을 찍고 사용하는 함수입니다.

2. 조건문(Condition) 관련 핵심 용어
if 문: 조건이 **참(True)**일 때 실행되는 문법입니다.

else 문: if 조건이 **거짓(False)**일 때 실행되는 문법입니다.

elif 문: 여러 개의 조건을 검사할 때 사용하며, **else if**의 줄임말입니다.

불리언 (Boolean): 조건문의 결과값인 **True**와 **False**를 통칭하는 데이터 타입입니다.

들여쓰기 (Indentation): 코드의 구분을 위해 사용하는 4칸의 공백입니다. 파이썬에서는 매우 중요합니다.

비교 연산자: ==(같다), !=(다르다), >=(크거나 같다) 등 값을 비교하는 기호입니다.

논리 연산자: and(모두 참), or(하나라도 참), not(반대로) 등 조건을 조합하는 단어입니다.

3. 반복문(Loop) 관련 핵심 용어
for 문: 리스트나 범위 안의 요소를 하나씩 꺼내어 반복할 때 사용합니다.

in 키워드: for 변수 in 바구니 형태로 쓰이며, 꺼낼 대상이 어디 있는지 지정합니다.

range() 함수: 특정 횟수만큼 반복하고 싶을 때 숫자 범위를 만들어주는 함수입니다. (예: range(5)는 0부터 4까지)

4. 자료구조 관련 핵심 용어
리스트 (List): [ ] 대괄호를 사용하며, 여러 값을 순서대로 담는 바구니입니다.

딕셔너리 (Dictionary): { } 중괄호를 사용하며, **키(Key)**와 **값(Value)**이 한 쌍으로 이루어진 구조입니다.

인덱스 (Index): 리스트 등에서 데이터의 위치를 나타내는 번호이며, 0번부터 시작합니다.

Q: 함수 내부의 결과를 밖으로 내보낼 때 사용하는 키워드는?

A: return (반환)

Q: if문이 거짓일 때 다른 조건을 추가로 검사하고 싶다면?

A: elif

Q: 함수를 정의할 때 입력값을 받기 위해 선언하는 변수의 이름은?

A: 매개변수 (Parameter)

Q: 조건문이나 함수 선언 줄 마지막에 반드시 붙여야 하는 기호는?

A: 콜론 (:) -->
